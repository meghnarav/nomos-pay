# Nomos Pay
A secure payment and ledger system.

## Features
- Double Entry Ledger
- Idempotency Keys
- Transactional Locks
- Audit Logs

## Tech Stack
- Language: Java (Spring Boot)
- Database: PostgreSQL
- Cache: Redis
- Tool: Postman
